# tornado-annotated

- tornado 源码 注解

## 项目说明:

1. 对 tornado 框架 源码,添加学习注释
2. 目前选择的版本:
    - 1.0.0    这是 tornado 的第一版,代码应该是最简洁的
    - 4.2.1    简单对github 上 使用 tornado 框架的 版本,进行初步统计,发现使用该版本的项目最多.
    - 故不再选择 2.x, 3.x 的版本 学习注解
     

## 本项目结构:

- 会包含tornado框架的结构剖析.
    - 包括 思维导图等工具制作的笔记
    - 参考阅读文档
    - tornado 核心模块源码重点剖析
